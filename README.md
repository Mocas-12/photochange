# PhotoChange 使用说明

一个简洁的前端图片处理工具：上传、裁剪、调整尺寸、格式转换（PNG/JPG/PDF）。全部在浏览器本地完成，无需安装、无需上传到服务器。

## 功能一览

- 上传方式：按钮选择或拖拽到页面。
- 裁剪：自由/1:1/4:3/16:9/3:2，支持“裁剪选区”“重置”。
- 自由裁剪时：选区内部透明、选区外部半透明遮罩，便于观察。
- 尺寸：填写宽高，支持“锁定比例”，点击“应用尺寸”。
- 导出：PNG/JPG/PDF。
  - JPG 支持质量调节（0.1–1），数值越小文件越小。
  - PDF 基于 A4 页面，图片自适应并居中。
- 文件名：可在下载前输入文件名。
- 状态徽章：右下角实时显示图片分辨率/格式/文件大小等信息。

## 使用步骤

1. 打开网页（或本地预览）。
2. 点击“选择图片”或直接拖拽图片到页面。
3. 在画布上按住拖动创建选区；必要时切换长宽比。
4. 点击“裁剪选区”完成裁剪，或“重置”回到原图。
5. 在“尺寸”面板填写宽/高，勾选“锁定比例”可等比缩放，点击“应用尺寸”。
6. 在“导出”面板选择 PNG/JPG/PDF；若为 JPG，可调整“质量”。
7. 输入文件名，点击“下载”。

## 常见问题

- 为什么 PNG 没有“质量”选项？
  - PNG 为无损压缩，通常不提供有损“质量”参数；文件大小主要由图片内容决定。
- PDF 的清晰度如何？
  - 实际是把当前画布转为 JPG/PNG 嵌入到 PDF 中；清晰度取决于画布尺寸与 JPG 质量。
- 下载文件过大怎么办？
  - 先降低 JPG 质量，或先在“尺寸”面板缩小分辨率再导出。
- 隐私与安全？
  - 全部在浏览器本地处理，不会上传到服务器。

## 本地预览

无需安装依赖，直接用任意静态服务器或浏览器打开 `index.html` 即可。推荐方式：

```bash
python -m http.server 5173
# 浏览器访问 http://localhost:5173/
```

## 备注

- 依赖：`jsPDF` 通过 CDN 加载，在线访问可直接使用；如需完全离线，可将 `jsPDF` 下载到本地并把 `index.html` 的引用改为本地文件。
